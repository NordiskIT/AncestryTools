Privacy note

All tools process files in your browser. GEDCOM and derived results remain on your machine unless you explicitly save or share them. There are no remote uploads, analytics or tracking included by default.

To preview locally: run python -m http.server 8000 or use a similar static server and open http://localhost:8000.
